# [Trade Insights Dashboard](https://g-trade.vercel.app/)

## Overview
This project, built with Next.js and PostgreSQL, offers a comprehensive analysis of a country's exports and imports from January to November 2023. The Trade Insights Dashboard aims to provide users with valuable information about trade trends, top trading partners, and the overall dynamics of the country's market.

## Features
- Dashboard Overview: Visual representation of key trade metrics.
- Trade Partners: Analysis of top trading partners and their market share.
- Monthly Reports: In-depth examination of monthly trade performance.
- Top Trade Products: Identification of the most imported and exported product categories.
- Countries Page: Overview of each country's total exports and imports.
- Country Pages: Detailed breakdown of top export and import products for each country.

- Next.js: A React framework for building efficient and scalable web applications.
- PostgreSQL: A powerful open-source relational database for data storage.

## Usage
- Explore the various sections of the dashboard to gain insights into the country's trade dynamics.
- Visit the "Countries" page to get an overview of each country's total exports and imports.
- Click on individual countries to access detailed information about their top export and import products.
